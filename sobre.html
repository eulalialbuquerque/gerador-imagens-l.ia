Sobre o Projeto

O que é o Gerador de Imagens com IA?

Este projeto foi desenvolvido como parte da disciplina de Introdução a Computação, com o objetivo de criar uma aplicação que utiliza inteligência artificial para gerar imagens a partir de descrições textuais.

A aplicação demonstra a integração de tecnologias web modernas com APIs de IA, permitindo aos usuários explorar o potencial criativo da inteligência artificial na geração de conteúdo visual.

Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

  • HTML5: Para estruturação do conteúdo
  • CSS3: Para estilização e layout responsivo
  • JavaScript: Para interatividade e integração com APIs
  • APIs de IA: Para geração de imagens a partir de texto
